# Alarme Inteligente com BitDogLab

![Imagem do Projeto](https://github.com/fabiofonteles1/Embarcados/blob/main/projeto%20final%20embarcados/png/Captura%20de%20tela%202025-02-08%20221404.png)  

Descrição

Este projeto consiste no desenvolvimento de um alarme inteligente utilizando exclusivamente a placa BitDogLab. O sistema usa os recursos internos da placa para monitoramento e controle local de segurança.

Funcionalidades

Alerta sonoro via buzzer

Indicação visual com LEDs

Registro de eventos na memória interna da BitDogLab

Monitoramento de entrada digital para acionamento do alarme

## Hardware Utilizado
- **Placa:** BitDogLab
- **Atuadores:** Buzzer, LEDs
- **Alimentação:** via USB

## Configuração e Execução
1. Clone este repositório:
   ```sh
   git clone https://github.com/fabiofonteles1/Embarcados.git
   ```
2. Carregue o firmware na BitDogLab.
3. Compile e grave o código na placa.
4. Monitore o funcionamento por meio de LEDs e alertas sonoros.
